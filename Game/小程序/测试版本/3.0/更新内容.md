新增功能
1. 杂交培育系统
入口：农场页面顶部「开启杂交模式」按钮
操作：开启后点击两块相邻的成熟作物，自动执行杂交
10 种杂交配方，按等级解锁：
配方	结果	解锁等级
胡萝卜 + 番茄	彩椒萝卜	Lv.2
番茄 + 小麦	红麦	Lv.2
玉米 + 小麦	黄金麦穗	Lv.3
胡萝卜 + 玉米	甜玉萝卜	Lv.3
草莓 + 番茄	樱桃番茄	Lv.4
玉米 + 南瓜	金瓜玉米	Lv.5
草莓 + 西瓜	蜜瓜草莓	Lv.5
向日葵 + 番茄	阳光番茄	Lv.5
小麦 + 西瓜	水晶麦粒	Lv.5
南瓜 + 向日葵	金阳南瓜	Lv.6
杂交基础成功率 70%，采矿技能 Lv.4 可提升至 95%
失败则两块作物均被消耗

2. 技能树系统（4 条分支，各 5 级）
技能	效果
农耕	生长速度 +10%/+15% → 收获金币 +15% → 经验 +30% → 价格翻倍
钓鱼	每日金币 +10/+15 → 收获 20%/35% 概率翻倍 → 额外 +50 金币
采矿	种子折扣 10%/20% → 种植经验 +5 → 杂交成功率 +25% → 种子免费
烹饪	浇水效果 +1/+1 秒 → 生长时间 -15%/-15% → 瞬间成熟
每次升级获得 1 个技能点
技能页面在底部 TabBar「技能」入口
3. 技能加成已集成到核心玩法
种植时自动应用生长加速、种子折扣、种植经验
收获时自动应用金币加成、双倍概率、经验加成
浇水效果受烹饪技能影响
商店价格受采矿技能折扣影响
